## Auto Teleport

A mod that automatically teleports the player who sent the /locate command.

## Usage

Instead of opening chat, hovering over the coordinates, and sending the teleport command after executing the command, simply use the /locate command. This mod significantly speeds up the process of finding structures or biomes, as it requires fewer actions. Additionally, a message with the player's previous coordinates is sent to chat, making it easy to return to your previous location.
